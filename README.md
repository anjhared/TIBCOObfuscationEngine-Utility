# TIBCOObfuscationEngine-Utility
This is a tool which has the purpose to allow decryption/deobfuscation of TIBCO BW 5.x encrypted/obfuscated strings. In addition, you can use this tool to encrypt/obfuscate readable passwords.

DISCLAIMER: You will not find any TIBCO software in this repository. In order to use this tool you will need to have TIBCO BW 5.x already installed in your machine. This is only a helpful tool to get around the blocker of a forgotten or unknown password.
